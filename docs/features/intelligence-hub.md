# Intelligence Hub — Feature Doc

**Status:** Planned | **Skill:** `/toolkit research`

## Overview

A self-contained HTML dashboard with 7 tabs of actively researched business intelligence. Generated by the `/toolkit research` skill, it serves as the product owner's single source of truth for market data, competitive positioning, pricing strategy, and go-to-market planning.

## Detailed Design

See planning docs:
- `docs/plans/toolkit-research-flow.md` — Full 9-step execution flow
- `docs/plans/html-prototype-spec.md` — CSS architecture, components, JS patterns

## Key Decisions

| Decision | Choice | Why |
|----------|--------|-----|
| Output format | Self-contained HTML | Works offline, no server, opens in any browser |
| Charting | Chart.js (embedded) | Single-file, no CDN dependency for offline use |
| CSS | Custom properties + minimal classes | Themeable, no framework dependency |
| Framework | Vanilla JS | No build step, any developer can read/modify |
| Data | Hardcoded at generation time | No API calls at runtime, works forever |
